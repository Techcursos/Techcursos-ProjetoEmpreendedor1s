    FRONT-END
    Nosso foco é tornar você um desenvolvedor de Front-End, mas afinal o quê é front-end?
    Front-end é responsável pela interação visual e funcionalidade de um site ou aplicativo.
    Nesses requisitos, o programador utilizará as linguagens de programação associadas a HTML, Css e JavaScript.
    HTML
    No grosso modo, exemplificando as funcionalidades de cada linguagem, o Html será responsável pela estruturação de uma página, como textos, rodapé, cabeçalho, menus, barra etc.
    CSS
    Já na parte do css, páginas em estilo cascata, do inglês, Cascading Style Sheets, será usado os recursos em códigos para mudar a cor, tamanho, imagens, efeitos e estilos. Podemos dizer que o css completa o Html, pois juntos, como se fosse a construção de uma casa, o HTML como estrutura e o css como pintura e modelagem.
    JAVASCRIPT
    Essa linguagem, atrelada com as outras, formam a base do front-end. O JavaScript é responsável pela funcionalidade e comportamentos de um site, deixando-o mais dinâmico e interativo para o usuário. É importante destacar que JavaScript não é a mesma coisa que Java, são parecidos, mas totalmente diferentes as suas aplicações.

    Agora que você já entende um pouco sobre front-end, é importante destacar que a tecnologia está aí para facilitar nosso dia a dia, seja numa página de compra online ou até mesmo num site de notícias.

    O que é Lógica de programação

    Lógica de programação é a técnica usada para encadear pensamentos para atingir determinado objetivo. Estes pensamentos, podem ser descritos como uma forma de instruções, que devem ser seguidas para se cumprir uma determinada tarefa. Sequência lógica são passos executados até atingir um objetivo ou solução de um problema. Um dos diferenciais da lógica de programação é a técnica de solucionar problemas de forma rápida e lógica, assim trazendo um melhor resultado para os seus problemas.
